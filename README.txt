= lolladolla cryptocurrency =

Pronounced "law-luh daw-luh", after "Lawlor Dollar".
Symbol: ₤ (lira, &#8356;)

Purpose: CS 493 midterm exam


== Protocol Description ==

A transaction transfers ₤ from one account to another.

Transactions are signed by the originating account.


== Software Description ==
Set up dependencies:

pip3 install circuits


Start the server:

python3 lolladolla.py


Surf to:
	http://localhost:8080
to make transactions!
